1000-Introduction


INTRODUCTION TO SYMCOMMANDER

SymCommander is a file manager for SymbOS, a program in the style of the classic Norton Commander to copy, move or delete files. SymCommander can do much more like surfing through directory trees, displaying detailed file and drive information, changing file attributes, split and combine files etc.

SymCommander can be controlled by mouse as well as by keyboard. There are keyboard shortcuts for all important operations, which speeds up the usage of SymCommander a lot.

2000-Using SymCommander

2100-Screen Layout


SCREEN LAYOUT

SymCommanders screen mainly contains two large directory lists on the left and the right side. Each shows the folder content of a selected drive and path, which are shown above the list. The path of the active list is displayed inverted. This is also the source directory for file operations, the non-active list is always the destination directory. At the bottom of each list you will see a summary of the number and total sizes of all and the selected files and directories.

At the bottom of the screen you will find a textinput line for entering commands directly like in a text shell. Next to it there are the buttons for some of the most important file operations. Like many SymbOS applications SymCommander contains a menubar at the top of the window, were you find all available functions.

2200-Pull Down Menus

2210-Files


"FILES" MENU

>Open

Opens or executes the selected file. If you typed something into the commandline the system will try to start this file including all attached parameters. Otherwise it will load the one selected in the current list. EXE and COM files will be executed directly, while documents will be loaded with their associated application.

>Change Attributes

This opens a dialogue, which allows you to set or reset any attributes of one or multiple selected files and directories at the same time in just one step. You can also change their timestamps. Choose "unchng" for any attribute types if you want to leave them unchanged.

>Properties

In this dialogue you will receive several information about on single selected file or directory, like its date and its side. It also allows you to rename it and modify its attributes.

>Drive Information

Displays all data of the current disc drive or harddisc partition. This included the drive and filesystem as well as the total amount of free and used space.

>Split File

If you want to split one large file into an amount of smaller files (which may fit on a floppydisc then) you can use this function. You have to specify the size and the destination.

>Combine Files

To unite any amount of split files into one large file again you have to use this function. It will copy all selected files into one big file again.

2220-Mark


"MARK" MENU

The mark menu is for quick selecting of an amount of files and directories inside the current list. You can then do any operations on the selected files (see FILE or COMMANDS menu).


>Select Group

This allows the selection of all files with a name and/or extension, which are matching to a specified file mask. Allowed wildcards are ? (any single char) and * (any chars until the end of the name section).

>Unselect Group

This is working in the same way like "Select Group" but it will unselect all matching files.

>(Un)Select All

Selects or unselects all files and directories in the current list.

>Invert Selection

Inverts the current file and directory selection. All selected items will be unselected and all unselected will be selected.

[this feature is not yet implemented]

2230-Commands


"COMMANDS" MENU

>Copy Files

Copies all selected files to the destination drive and directory. Already existing files at the destination can be optionally overwritten or skipped.
[recursive subdirectory handling is currently not supported]

>Move Files

Renames or moves files. If no file is marked, the current selected file is renamed. If one or more files are marked they will be moved to the destination directory.
[recursive subdirectory handling is currently not supported]

>Delete Files

Deletes all selected files.

>New Folder

Creates a new subfolder inside the current directory.

>View/Edit/Search

[these features are not yet implemented]

>Source <-> Target

Swaps the path of the left and the right directory list.

>Target = Source

Sets the path of the current directory for both left and right lists.

2240-Show


"SHOW" MENU

The show menu is for modifying and filtering the view of the currently displayed directory.


>All Files/Custom

SymCommander allows you to limit the directory display according to a special file mask. You will only see files, which names are matching to the file mask you specified. Choose "Custom" and enter the desired file mask. Allowed wildcards are ? (any single char) and * (any chars until the end of the name section).

For deactivating the file mask filter choose "All files". To reactivate the last filter again choose the second menu item.

>Sort by

You can sort the directory listing by file name, file size or file date by choosing one of these three menu items. You can also click the associated title of the directory list to change the sorting. By clicking multiple times on the list titles the sorting will switch between ascending and descending.

>Reversed Order

This will change the current sorting between ascending and descending.

>Reread Source

This will reload and display the directory of the current drive and path of the active list again. Use this if you changed the floppy disc or if other applications made changes to files and directories.

2250-Configuration


"CONFIGURATION" MENU

>Options

[...]

>Save Settings

This will save the current settings. The path of the left and the right directory lists are saved as well as the list and buffer parameters and additional settings of the option dialogue described above. They will be restored on the next start of SymCommander.

The settings are saved in the SYMCMDER.INI file, which is located in the same drive and directory like SymCommander (SYMCMDER.EXE) itself. If you are working with a floppy-based system please take care, that the correct disc containing SymCommander is inserted when saving the settings.

2300-Keyboard shortcuts


KEYBOARD SHORTCUTS

The following is a list of all available keyboard shortcuts.

Navigation

Tab switch between left/right directory
Del jump to parent directory
Alt+F1 change left drive
Alt+F2 change right drive
move through the current list
+Ctrl jump through the current list
Ctrl+R reload directory of the current list
Ctrl+U swap left and right list
Escape delete command line text
Return open selected or entered file

File selection
(see also MARK menu)

Space (un)select files in the current list
Shift+ (un)select files in the current list
* invert the current file selection
+ add files of a special type
- remove files of a special type

File operations
(see also COMMANDS and FILES menu)

F1 open help
F2 show drive information
F5 copy selected files
F6 move or rename selected file(s)
F7 create new sub directory
F8 delete selected files
Alt+Ret edit properties of the selected file
